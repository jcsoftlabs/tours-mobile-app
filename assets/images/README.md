# Assets Images

Placez ici les images de l'application :
- Logos
- Images d'établissements
- Icônes personnalisées
- Images de fond

## Formats recommandés
- PNG pour les transparences
- JPG pour les photos
- SVG pour les icônes vectorielles (à convertir en PNG pour Flutter)

## Tailles recommandées
- Logo app : 1024x1024 px
- Images cards : 300x200 px
- Icônes : 24x24, 48x48, 72x72 px
